Welcome

Welcome to AQUA-MER, a database and toolkit for researchers working on environmental mercury geochemistry.

AQUA-MER is useful for researchers who want to know how mercury will speciate in defined environments i.e. to which chemical elements and groups it will bind and in what proportions, given an input of concentrations of chemicals.

AQUA-MER
    a) Contains a database of thermodynamic associations of mercury in aqueous solutions.
    b) Calculates speciation profiles as a function of pH given input concentrations of chemicals, using the well-known program, PHREEQC, from U.S. Geological Survery (USGS).

Background.
In order to calculate speciation profiles it is necessary to know the association constants (log K) of mercury with the chemicals present. AQUA-MER is compiling a list of association constants from three sources:
    a) Experimental, where available.
    b) Previously calculated, using a validated quantum chemical methodology.
    c) In cases where neither a) nor b) exist AQUA-MER provides a toolkit that researchers can use to calculate the values themselves.

Before proceeding, please make sure you have A) the list of chemicals in your environmental sample B) their concentrations.
