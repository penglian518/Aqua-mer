Welcome
AQUA-MER is a database and toolkit for researchers working on environmental mercury geochemistry.

AQUA-MER
    a) Calculates speciation profiles given input concentrations of chemicals.
    b) Contains a database of thermodynamic associations of mercury in aqueous solutions.

Researchers come to AQUA-MER if they want to know, given a certain input of concentrations of chemicals in the environment, how mercury will speciate (what it will bind to). In order to calculate speciation it is necessary to know the association constants (log K) of mercury with the chemicals present. AQUA-MER is compiling a list of association constants from experimental sources. It has also values calculated in the UTK/ORNL Center for Molecular Biophysics (CMB) using a validated quantum chemical methodology. In cases where the log K values are neither known experimentally nor have been calculated, AQUA-MER provides a toolkit that researchers can use to calculate the values themselves.
