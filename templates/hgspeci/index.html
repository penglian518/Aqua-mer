Instruction on how to use the Speciation module

The Speciation module provides an intuitive online tool for researchers to study the speciation of given solutions. Although it integrate extensive parameters for Hg related complexes, it can also be applied to study the speciation of other elements, such as Fe, Cd, Co etc. It allows the researchers to study the solutions with their own parameters as well. Together with the online toolkit provided by AQUA-MER, the users can easily calculate the necessary parameters with quantum mechanis methods, such as DFT.

To use the Speciation module, the following three steps are required: 1, input the parameters; 2, review the input and submit the job; 3, view and download the results.
If you are familiar with AQUA-MER, you can start by clicking the "Input parameters" from the left menu. Otherwise, please read this instruction first.

Step 1. Input parameters.
In this step, the user will build up a vertual solution which contians the Environmental Conditions and the Individual Element/species. Parameters in the Environmental Condistions section, such as Temperture, pH range, Conenctration Unit etc. will be applied system wide. Parameters in the Individual Element/species section will only affect the corresponding element/species and will override the parameters in the first section.
    a. Environmental Conditions parameters (units, pH range, pe, density, temperatures, redox ...)


    b. Individual Element/species parameters (will provide related parameters in the database, and the option to add parameters by the users.)
Tips: While inputting the elements, it will searching the database automatically. If the related parameters are available, it will display them in the bellow. Otherwise, the user can provide their own parameters.


Step 2. Review & submit. Before submission, you can review your input parameters and parameters from database.


Step 3. View and download the results. Once the job is finished, you can view your results and download them.

Notice: Once the compound is uploaded, a unique job ID will be assigned. With the job ID, you can check your results and redo analysis quickly.
